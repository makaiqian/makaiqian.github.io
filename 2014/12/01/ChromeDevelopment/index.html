Console

  1. 几种
    console.log()
    console.error()
    console.warn()
    console.assert() 条件判断，两个参数，如果第一个参数为false，则第二个输出。
    console.group() 形成一组，可嵌套。
    console.groupEnd()
    console.dir() 输出javascript描述
    console.time() 时间，要求参数里面输入的都是同一字符串。
    console.timeEnd()
    console.timeStamp() 时间线录制时起作用
    debugger; 断点
    monitorEvents() 监听事件 monitorEvents(对象,事件)
    unmonitorEvents() 停止监听 unminitorEvents(对象)
    profile() 分析器，可嵌套 可以在Profiles里面看。
    profileEnd()

  2. 清除console

  • 右键
  • clear()
  • console.clear()
  • cmd + K

  1. 删选console输出的信息

  2. 字符串替换
    %c 样式
    （其余和c语言类似）

  3. 选取对象
    $() 元素 querySelector
    $$() 数组 querySelectorAll
    $x(‘/html/body/script’) 数组
    $0-$4 最近到最远的5个对象

快捷键(Mac下)

cmd + opt + i / j 打开／关闭开发者工具
cmd + o 通过文件名查找
cmd + shift + o 根据函数或者选择器查找
cmd + f 当前文件中查找文本
cmd + opt + f 所有已加载的文件中查找
esc 切换drawer
shift + enter 换行
